Simple Facbook header generator

== Description ==

Add facebook tags to header
- Title
- Description
- Image
- AppID
- PageID

== Installation ==


1. Upload `finger.php` to the `/wp-content/plugins/` directory
2. Upload `fingersimplefacebookheader.php.php` to the `/wp-content/plugins/` directory
3. Upload `admin.fingersimplefacebookheader.php` to the `/wp-content/plugins/` directory
4. Upload `class.fingersimplefacebookheader.php` to the `/wp-content/plugins/` directory
5. Activate the plugin through the 'Plugins' menu in WordPress
